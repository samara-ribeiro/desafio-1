TUDO SOBRE

Definição

A linguagem de marcação de hipertexto (HTML) é uma linguagem de marcação de hipertexto, a linguagem de marcação padrão para documentos projetados para serem exibidos e visualizados on-line durante um navegador também ajuda a criar a estrutura da página da web. por ser uma linguagem de marcação, consiste em muitas tags. Existem tags para exibir texto, tabelas, listas ordenadas e listas não ordenadas, etc.
HTML5 é uma linguagem de marcação de tecnologia central da Internet usada para estruturar e apresentar conteúdo para a World Wide Web. Em outubro de 2014, esta é a quinta revisão final e completa do padrão HTML do World Wide Web Consortium (W3C). A versão anterior, HTML 4, foi padronizada em 1997.
Também vale notar que o HTML agora é considerado um padrão oficial da internet. O World Wide Web Consortium (W3C) mantêm e desenvolve especificações do HTML, além de providenciar atualizações regulares.

Finalidade

Existem duas seções principais na página HTML: seção de cabeçalho e corpo. Os dados que descrevem a página também denominada como metadados estão dentro da seção head, enquanto a seção body inclui todas as tags que são necessárias para representar o conteúdo visível da página da web HTML é uma linguagem independente de plataforma para que possa ser usada em qualquer plataforma como Windows, Linux, Macintosh, etc.
Existem várias versões HTML. A versão mais recente é o HTML 5. São recursos mais avançados, como geolocalização, áudio nativo e suporte a vídeo, Canvas, web socket, etc. Normalmente, o HTML é uma linguagem simples de descobrir e usar. Um programador pode criar um arquivo HTML empregando um editor de texto simples e executá-lo utilizando um navegador.
